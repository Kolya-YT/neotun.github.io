Возможности

Что даёт NeoTUN

Сервис, а не очередной клиент. Подключаетесь через привычные приложения.

Быстрая настройка

Подписка подтягивается автоматически. Настоящая настройка занимает меньше минуты.

Разные платформы

Windows, Android, Linux, macOS и iOS. Одна подписка — на всех устройствах сразу.

Несколько стран

Серверы в разных регионах. Переключайтесь между локациями в пару кликов.

Простое обновление

Продлить подписку можно в один тап. Ключи и серверы обновляются без ручной настройки.

Бесплатный тариф

Попробуйте сервис бесплатно. Без карты и без ограничений по функциям.

Поддержка в Telegram

Ответим по существу в течение нескольких минут — без ботов-шаблонов.

FAQ

Частые вопросы

Не нашли ответ — напишите в поддержку

NeoTUN — это сервис защищённого доступа, который работает с совместимыми открытыми клиентами. У NeoTUN нет собственного приложения: вы подключаетесь через сторонний клиент, добавив в него подписку NeoTUN.
INCY, Happ, Hiddify, Sing-box, v2rayNG, v2rayN, Nekoray и другие открытые клиенты. Все приложения скачиваются напрямую из официальных GitHub-релизов. Полный список — на странице «Приложения».
Получите подписку через Telegram-бот, выберите совместимое приложение на странице загрузок, добавьте подписку и нажмите «Подключить».
Принимаем оплату через СБП, CryptoBot и Heleket. Также можно пополнить баланс и оплачивать с него.
Одна подписка работает на неограниченном количестве устройств одновременно. Подключайте телефон, ноутбук и планшет без доплат.
Приложения разрабатываются независимыми авторами и распространяются через официальные GitHub-релизы. NeoTUN ведёт на оригинальные файлы без изменений — так вы получаете проверенные сборки.
